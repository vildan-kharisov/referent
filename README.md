# Referent 
Референт — переводчик с ИИ-обработкой

## Описание

Приложение для анализа англоязычных статей с использованием AI:
- Парсинг статей (извлечение даты, заголовка, контента)
- Перевод статей
- Генерация описаний, тезисов и постов для Telegram

## Быстрый старт

### Установка зависимостей

```powershell
pnpm install
```

### Настройка API ключей

Создайте файл `.env.local` в корне проекта и добавьте API ключи. Подробные инструкции в [SETUP.md](./SETUP.md).

**Минимальная настройка (YandexGPT):**
```env
YANDEX_GPT_API_KEY=ваш-ключ
YANDEX_FOLDER_ID=ваш-folder-id
```

### Запуск

```powershell
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

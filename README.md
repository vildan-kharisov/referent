## Referent — минимальное приложение на Next.js

В этом репозитории создано минимальное приложение на Next.js (app router, TypeScript).

- **Разработка**: `npm run dev`
- **Сборка**: `npm run build`
- **Продакшн-сервер**: `npm run start`

### Установка и запуск (PowerShell)

В корне проекта выполните в PowerShell:

```powershell
npm install
npm run dev
```

После этого откройте в браузере `http://localhost:3000`.

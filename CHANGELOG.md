# Changelog

Все значимые изменения в проекте документируются в этом файле.

## [1.0.0] - 2024

### Добавлено

#### Основной функционал
- Парсинг англоязычных статей (извлечение даты, заголовка, контента)
- Перевод статей с английского на русский
- Генерация краткого описания статьи (2-3 предложения)
- Извлечение ключевых тезисов (5-7 пунктов)
- Генерация постов для Telegram

#### Инфраструктура
- Утилита для работы с AI-провайдерами (`app/lib/ai.ts`)
  - Поддержка YandexGPT (основной провайдер)
  - Поддержка OpenRouter (fallback)
  - Автоматический fallback при недоступности основного провайдера
  - Retry-логика с exponential backoff
  - Разбиение длинных текстов на части

#### Оптимизация
- In-memory кэширование результатов AI (TTL: 1 час)
- Автоматическая очистка устаревших записей кэша
- Обработка длинных статей (>8000 символов) с автоматическим разбиением

#### UX улучшения
- Кнопка копирования результата в буфер обмена
- Визуальная обратная связь при копировании
- Улучшенное отображение ошибок с контекстными инструкциями
- Индикаторы прогресса для длинных операций

#### Документация
- README.md с полным описанием проекта
- SETUP.md с инструкциями по настройке AI-провайдеров
- API.md с документацией API и примерами
- PLAN.md с планом реализации
- PROJECT.md с описанием проекта

### Технические детали

- Next.js 14 с App Router
- TypeScript для типизации
- Tailwind CSS для стилизации
- Cheerio для парсинга HTML
- Поддержка нескольких AI-провайдеров с fallback

### Известные ограничения

- Кэширование работает только в рамках одного процесса (для продакшена рекомендуется Redis)
- Максимальная длина статьи: ~50,000 символов (автоматическое разбиение)
